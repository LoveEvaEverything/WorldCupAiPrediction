# ⚽ World Cup AI Prediction System

Most people watch the World Cup.

I created a markdown file so AI can watch it too. 🤖

This project contains a structured prompt framework that guides AI through a professional football prediction workflow.

Instead of making guesses, the AI evaluates:

* FIFA Rankings
* ELO Ratings
* Recent Team Form
* Squad Availability
* Injuries & Suspensions
* Team News & Manager Comments
* xG Metrics
* Betting Market Movements
* Historical Matchups

The goal is simple:

**Make football predictions using data, not emotions.**

## 🧠 How It Works

The prompt instructs AI to:

1. Discover today's World Cup matches
2. Gather team statistics
3. Analyze player availability
4. Review recent news
5. Compare tactical strengths
6. Calculate probabilities
7. Generate match predictions
